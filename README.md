### 常规版(master)使用

```bash
# 克隆项目
git clone https://github.com/vue-admin-beautiful/admin-plus.git
# 进入项目目录
cd admin-plus
# 安装依赖
npm i
# 本地开发 启动项目
npm run serve
```

### 其他版本下载

`${branch_name}`替换成上表分支名即可

```bash
# 克隆项目
git clone -b ${branch_name} https://github.com/vue-admin-beautiful/admin-plus.git
# 进入项目目录
cd admin-plus
# 安装依赖
npm i
# 本地开发 启动项目
npm run serve
```
